ELT Pipeline with DLT & dbt (REST API → Analytics Marts)

Built an end-to-end ELT pipeline that extracts data from multiple REST API endpoints using DLT, loads it into PostgreSQL, and transforms it into analytics-ready fact and dimension tables using dbt. The pipeline follows a layered Bronze → Silver → Gold modeling approach with robust testing, documentation, and data quality enforcement.

The solution supports analytical use cases such as user engagement, content performance, task completion behavior, and cohort-style analysis, enabling BI tools to consume clean, well-modeled datasets.

Key Concepts: Analytics Engineering · Data Engineering · DLT · dbt · ELT · REST APIs · PostgreSQL
